## Program Structure

The Executive Leadership Lab spans 10 months with monthly modules:

| Month | Module | Focus |
|-------|--------|-------|
| January | Module 1 | Break Through Career Plateaus |
| February | Module 2 | Master Your Executive Presence |
| March | Module 3 | Build Your Strategic Brand |
| April | Module 4 | Navigate Bias and Lead Through It |
| May | Module 5 | Amplify Your Voice |
| June | Module 6 | Build Your Board of Advisors |
| July | Module 7 | Lead and Develop Teams |
| August | Module 8 | Lead Difficult Conversations |
| September | Module 9 | Get Promoted |
| October | Module 10 | Set the Table for Your Future |

## Design System

- **Typography:** Source Serif 4 (headings), Inter (body)
- **Colors:** Charcoal (#1A1A1A) + Teal accent (#3EBCE8)
- **Aesthetic:** Executive, minimal, professional

## Features

- Click-to-advance navigation with arrow buttons
- Keyboard navigation (left/right arrows)
- Touch/swipe support for mobile
- Progress bar indicator
- Print-friendly layouts
- Fillable form fields

## Local Development

Clone and open any HTML file directly in a browser:

```bash
git clone https://github.com/mdxvision/cwc-executive-lab.git
cd cwc-executive-lab
open index.html
```
